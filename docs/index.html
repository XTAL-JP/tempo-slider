TEMPO Slider

CDJ-style tempo & pitch keep controls for music purchase sites.

Get it for Chrome Get it for Firefox View on GitHub Privacy Policy

What it does

Adds a CDJ-style tempo control panel to music purchase sites. Designed for DJs who want to preview tracks at a target BPM before buying.

  • Vertical TEMPO fader with ±6 / ±10 / ±16 / WIDE range, like a real CDJ
  • MASTER TEMPO (pitch keep) using DAW-grade Rubber Band Library
  • BPM display with automatic extraction on Beatport / Traxsource, or tap input
  • Keyboard shortcuts (,/., R, M, T) and mouse wheel
  • Draggable panel with position memory

Supported sites

  • Bandcamp — bandcamp.com
  • Beatport — beatport.com
  • Traxsource — traxsource.com
  • Plus any site you add via the popup

Privacy

No data is collected, tracked, or transmitted. All audio processing happens locally in your browser. See the Privacy Policy for details.
